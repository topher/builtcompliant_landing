# Built Compliant Landing Page

A modern, responsive landing page for Built Compliant - AI governance consulting services.

## Features

- **Professional Design**: Clean, modern interface optimized for B2B conversions
- **Responsive Layout**: Fully responsive design that works on all devices
- **Contact Form**: Interactive contact form with client-side validation
- **SEO Optimized**: Proper meta tags, structured data, and semantic HTML
- **Fast Performance**: Built with Next.js for optimal loading speeds

## Content Highlights

- Clear value proposition: "Governance That Closes Deals"
- Problem-focused messaging addressing real business pain points
- Three-tier service offering with clear pricing
- Professional contact form with compelling CTAs
- Trust-building elements and social proof

## Technology Stack

- **Framework**: Next.js 15 with App Router
- **Styling**: Tailwind CSS for responsive design
- **TypeScript**: Full TypeScript support for type safety
- **Deployment**: Optimized for Vercel deployment

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
